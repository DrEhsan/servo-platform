**Form Bot**

An demonstration for using Angular forms with Servo. Includes: 
* Angular service for communicating with Servo
* Submitting data fields from Angular to Servo
* Receiving and presenting fields from Servo

---

## Install

1. npm install
2. ng build
3. node server.js
